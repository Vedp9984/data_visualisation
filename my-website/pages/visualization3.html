Breed Size vs Lifespan Distribution

About Box Plots

A box plot shows the distribution of lifespans for each breed size group. Here's what each part means:

  • The box shows where 50% of the lifespans fall
  • The line in the middle of the box is the median lifespan
  • The whiskers show the full range of lifespans
  • The width of each box indicates the relative sample size

Key Observations

  • Hover over the boxes to see detailed statistics
  • Compare median lifespans across different size groups
  • Notice the range of lifespans within each size category
